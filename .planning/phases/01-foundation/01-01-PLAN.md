  <action>
    1. From `frontend/` directory, run: `npm install @tailwindcss/typography isomorphic-dompurify` and `npm install -D @types/dompurify`

    2. In `frontend/src/app/globals.css`, add the typography plugin AFTER the `@import "tailwindcss";` line:
       ```
       @plugin "@tailwindcss/typography";
       ```
       This is the Tailwind v4 syntax — do NOT use the v3 `plugins: []` array in a config file. The `@plugin` directive in CSS is how Tailwind v4 loads plugins.

    3. Create `frontend/src/lib/sanitize.ts` with a reusable DOMPurify wrapper:
       ```typescript
       import DOMPurify from "isomorphic-dompurify";

       export function sanitizeHtml(dirty: string): string {
         return DOMPurify.sanitize(dirty, {
           ALLOWED_TAGS: ["p", "br", "strong", "em", "ul", "ol", "li", "h2", "h3", "h4", "a", "span", "blockquote"],
           ALLOWED_ATTR: ["href", "target", "rel", "class"],
         });
       }
       ```
       This restricts allowed HTML tags to safe rich text elements only — no script, iframe, img (images come through assetUrl()), or form elements.

    4. In `frontend/src/components/product/ProductDetail.tsx` line 96, replace:
       ```
       dangerouslySetInnerHTML={{ __html: product.description }}
       ```
       with:
       ```
       dangerouslySetInnerHTML={{ __html: sanitizeHtml(product.description) }}
       ```
       Add import at top: `import { sanitizeHtml } from "@/lib/sanitize";`

    5. In `frontend/src/app/[locale]/noticias/[slug]/page.tsx` line 73, replace:
       ```
       dangerouslySetInnerHTML={{ __html: post.content }}
       ```
       with:
       ```
       dangerouslySetInnerHTML={{ __html: sanitizeHtml(post.content) }}
       ```
       Add import at top: `import { sanitizeHtml } from "@/lib/sanitize";`
